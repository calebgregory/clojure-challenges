# challenges

A place for me to learn clojure, doing challences from
[Project Euler](https://projecteuler.net) and
[4clojure](http://www.4clojure.com/problems)

## Usage

To make Java,

    $ java -jar challenges-0.1.0-standalone.jar [args]
